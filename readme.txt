Personnalisez dynamiquement l’URL du bouton “Poursuivre mes achats” affiché dans le mini-panier, via une option simple dans votre administration WordPress.

== Description ==

**Redirection Mini Panier** vous permet de rediriger le lien “Poursuivre mes achats” vers la page de votre choix : une catégorie, une page promotionnelle, ou votre accueil. Le tout sans toucher au thème ni au code.

Le plugin est conçu pour s’intégrer avec les blocs WooCommerce (mini-panier) chargés dynamiquement via JavaScript.

== Fonctionnalités ==

- Champ de réglage dans l’administration
- Redirection instantanée du lien dans le mini-panier
- Compatibilité blocs WooCommerce
- Léger, simple, activable à tout moment
- Aucun fichier du thème modifié

== Installation ==

1. Téléversez le plugin dans le dossier `wp-content/plugins/`
2. Activez-le dans le menu Extensions de WordPress
3. Allez dans **Réglages > Lien Mini-Panier**
4. Entrez votre URL personnalisée

== Frequently Asked Questions ==

= Le plugin fonctionne-t-il avec tous les thèmes ? =
Oui, tant que vous utilisez le bloc mini-panier de WooCommerce. Le plugin cible automatiquement sa structure.

= Puis-je désactiver la redirection facilement ? =
Oui, il suffit de laisser le champ vide dans les options du plugin. Aucune modification du site.
